# 麻將計分

四人麻將計分網頁，專為家庭聚會設計，支援手機直接使用。

🔗 **[立即使用](https://quilaty.github.io/Mahjong/)**

---

## 功能

- **東南西北座位** — 骰子隨機分配座位，開局更公平
- **風局追蹤** — 自動記錄東風東局至北風北局，莊家輪轉
- **連莊顯示** — 莊家連贏自動累計顯示（莊×2、莊×3…）
- **自摸 / 放槍** — 分別計算，放槍只有放槍者付款
- **底台 + 每台** — 直接設定元，每局自動計算金額
- **即時預覽** — 確認前先看每人輸贏金額
- **歷史紀錄** — 完整記錄每一局詳細資訊
- **最終結算** — 自動計算誰欠誰多少元

---

## 使用方式

### 開局設定

1. 輸入四位玩家小名（或按骰子隨機分配座位）
2. 設定底台金額（預設 50 元）與每台金額（預設 20 元）
3. 按「開始遊戲」

### 記錄每局

1. 按「記錄本局」
2. 選擇贏家 → 選台數 → 選自摸或放槍（放槍需再選放槍者）
3. 確認預覽金額後按「確認記錄」

### 結算

- 隨時按「結算」查看目前盈虧
- 顯示每人淨盈虧，以及誰應付給誰多少元

## 計分規則

| 胡法 | 付款方式 |
|------|---------|
| 自摸 | 三家各付（底台 + 台數 × 每台） |
| 放槍 | 僅放槍者付（底台 + 台數 × 每台） |

**風局輪轉：**
- 莊家贏 → 連莊
- 莊家輸 → 莊家移至下一位（東 → 南 → 西 → 北）
- 北家坐莊結束 → 場風進入下一圈（東風 → 南風 → 西風 → 北風）
